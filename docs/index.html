Built for Production AI Agents

From rapid prototyping to enterprise deployment. NOMOS scales with your needs.

Step-based Architecture

Define agent behavior as sequences of steps. Each step controls what tools are available and how decisions flow.

Multi-LLM Support

OpenAI, Mistral, Gemini, and more. Switch providers with configuration changes, not code rewrites.

Precise Tool Control

Control exactly which tools your agent can access at each step. No more unwanted function calls.

Configuration-Driven

YAML or Python configuration. Change agent behavior without touching core logic.

Session Management

Built-in persistence with Redis or PostgreSQL. Stateful conversations that survive restarts.

Deploy Anywhere

Docker containers, Kubernetes, AWS, Azure, GCP. Production-ready deployment options.
