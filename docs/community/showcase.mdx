## Submit Your Project

<Steps>
  <Step title="Build with NOMOS">
    Create an agent or application using the NOMOS framework
  </Step>
  <Step title="Document Your Work">
    Write a brief description, include screenshots, and share your code
  </Step>
  <Step title="Submit for Review">
    Open a PR or issue in our GitHub repository with your showcase submission
  </Step>
  <Step title="Get Featured">
    Approved projects will be featured on this page and in our newsletter
  </Step>
</Steps>

### Submission Guidelines

<AccordionGroup>
  <Accordion title="Technical Requirements">
    - Must use NOMOS framework as the core agent system
    - Include working demo or deployment link
    - Provide source code (open source preferred)
    - Include setup and usage instructions
  </Accordion>
  <Accordion title="Content Guidelines">
    - Write clear, engaging description (100-300 words)
    - Include 2-3 screenshots or demo GIFs
    - Highlight unique features and benefits
    - Mention tech stack and integrations used
  </Accordion>
  <Accordion title="Quality Standards">
    - Project should be functional and stable
    - Code should follow best practices
    - Documentation should be comprehensive
    - Should demonstrate creative use of NOMOS features
  </Accordion>
</AccordionGroup>
